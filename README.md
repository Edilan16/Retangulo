#  Retângulo

Projeto simples desenvolvido em **Java** para calcular propriedades de um retângulo a partir das medidas informadas pelo usuário.

O programa recebe a **largura** e **altura** do retângulo e calcula valores como **área e perímetro**, aplicando conceitos básicos de programação e matemática. A área de um retângulo é calculada multiplicando base pela altura (`A = base × altura`). ([lip-ee.github.io][1])

---

# Objetivo do projeto

Este projeto foi criado como exercício prático para reforçar conceitos iniciais de **Programação Orientada a Objetos (POO)** em Java.

Ele faz parte dos estudos de lógica de programação e estrutura básica de classes.

---

#  Funcionalidades

* Informar largura do retângulo
* Informar altura do retângulo
* Calcular área
* Calcular perímetro
* Exibir os resultados no console

# Como executar o projeto

1. Clone o repositório:

```
git clone https://github.com/Edilan16/Retangulo.git
```

2. Abra o projeto em uma IDE Java (Eclipse, IntelliJ ou VS Code).

3. Execute a classe principal do projeto.

4. Informe os valores solicitados pelo programa no terminal.

---

#  Exemplo de execução

Entrada do usuário:

```
Largura: 3.0
Altura: 4.0
```

Saída do programa:

```
Área: 12.0
Perímetro: 14.0
```
